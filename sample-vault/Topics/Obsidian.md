# Obsidian conventions

This server treats the vault as plain Markdown plus a few well-known
[[Topics/MCP|Obsidian]] conventions:

- **Wikilinks** look like `[[Other Note]]` or `[[Other Note|alias]]`.
- **Tags** are inline `#tag` tokens or a YAML `tags:` list in frontmatter.
- **Frontmatter** is a YAML block delimited by `---` at the top of a note.
- **Daily notes** live under `Daily/YYYY-MM-DD.md` by convention.
